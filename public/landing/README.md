# Landing page screenshots

Drop the 7 PNGs in this folder with these exact filenames. The landing page
references them by name and falls back to a placeholder if a file is missing.

| Filename | Used in section | What it shows |
|---|---|---|
| `dashboard.png` | Hero | Full TraderM8 dashboard — charts + bias cards + countdown |
| `sessions.png` | Spotlight: Bias & Session Levels | Asia / London / NY session cards + position calculator |
| `liquidity.png` | Spotlight: Liquidity Heatmap | NAS100 buy-side / sell-side liquidity pools |
| `calendar.png` | Spotlight: Economic Calendar | Monthly grid + Breaking News sidebar |
| `movers.png` | Spotlight: Market Movers | Top mover hero card + ranked list |
| `seasonality.png` | Spotlight: Seasonality | NQ April chart with 15y/10y/5y/YTD overlays |
| `journal.png` | Spotlight: Journal & Analytics | P&L calendar with green/red day cells |

Recommended export: PNG, ~2x the natural rendering size (e.g. 1600×1000) so
Retina screens stay crisp. Crop tightly to the panel — the landing page
wraps each image in its own browser frame.
